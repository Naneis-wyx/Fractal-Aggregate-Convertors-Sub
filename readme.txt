This program is a simple convertor.

Running program: After download use python 3 open the convertors.py 

Part 0: gas & temperature determine.


Part 1: calculate lamda based on pressure 
Output: lamda

Part 2: Choice 2 of 3 to import
Output: Size or Mass or Density

Part 3: Primary particle calculation
Output:N, Kundp, KunDve, KunDve2. Dve1 from formula:dve1=(6*m/pi/pho)^(1/3)
						    dve2= dpp*n(1/3)

Part 4: Cunningham slip correction factor
Output:Cc_Dp, Cc_Dve1, Cc_Dve2


General output chi(dve1) chi(dve2) chi(alternative)
